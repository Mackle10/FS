# Fellowship Schools Website (Eleventy + Netlify CMS)

Simple static site scaffolded with Eleventy and Netlify CMS for Fellowship Nursery School, Fellowship Primary School, and Fellowship High School.

Run locally:

```bash
npm install
npm run start
```

Build:

```bash
npm run build
```

Netlify CMS admin is available at `/admin/` when served with a Netlify identity/backend.
